# react-native-afterpay

React Native bridge for AfterPay SDK

## Installation

```sh
npm install react-native-afterpay
```

## Usage

```js
import Afterpay from "react-native-afterpay";

// ...

const result = await Afterpay.multiply(3, 7);
```
